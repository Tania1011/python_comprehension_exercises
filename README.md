# Python Comprehension Exercises

A collection of Python exercises for practicing loops, comprehensions, dictionaries, lists, and basic algorithms to strengthen Python programming skills.

---

## Description

This repository contains beginner-to-intermediate Python exercises covering:

- List comprehensions  
- Dictionary comprehensions  
- Nested dictionaries and loops  
- Conditional expressions (ternary)  
- Counting and aggregation (e.g., letter frequency, average salaries)  

Each exercise includes input data (lists, dictionaries, or sentences) and Python solutions.

---

## How to Run in Google Colab

1. Open [Google Colab](https://colab.research.google.com/).  
2. Click **File → Upload Notebook** and select the `.ipynb` file for the exercise you want to run.  
3. Run the code cells by clicking the **▶ Run** buttons or pressing **Shift + Enter**.  


---

## Key Concepts Practiced

- Python loops (`for`)  
- Conditional statements (`if`, `elif`, `else`)  
- Dictionary and list comprehensions  
- String manipulation and filtering (`isalpha()`, `lower()`)  
- Aggregations (`sum`, `len`)  
- Nested data structures (lists of tuples, nested dicts)
